¿que es jquery?


JQuery es una librería JavaScript muy liviana, que nos ayuda a usar mas fácilmente JavaScript en nuestros sitios web.

JQuery encapsula las tareas mas comunes que normalmente requieren muchas lineas de código para llevarlas a cabo, y  las entrega para que nosotros las llamemos con una sola linea de código. 


¿Para que sirve jquery?

  • Navegación y Manipulación del DOM
  • Manipulación de CSS (estilos)
  • Manipulación de eventos HTML
  • Ajax
  • Efectos y animaciones
  • y muchas funciones utilitarias

Que es el dom?


"Modelo de Objetos del Documento" o "Modelo en objetos para la Representación de Documentos" Proporciona un conjunto estándar de objetos para representar documentos HTML y XML.

El DOM es una interfaz de programación de aplicaciones para acceder, añadir y cambiar dinámicamente contenido estructurado en documentos con lenguajes como javascript.

Selectores css

Un selector CSS es un patrón usado para seleccionar los elementos a los cuales se les quiere dar estilo. ejem: 
#loginName // Selecciona un elemento con Id loginName .tituloGrande // Selecciona los elementos con clase tituloGrandeinput // Selecciona todos los elementos de tipo input:checked // selecciona todos los elementos chequeados
Jquery soporta todos los selectores CSS ademas de algunos propios.
:even // Selecciona todos los elementos pares en una selección:odd //Selecciona todos los elementos impares:visible //Jquery válida su visibilidad no su CSS display:button //Selecciona todos los input button o elementos button:animated: //Selecciona todos los elementos animados

¿Como sé si mi selector contiene elementos?

 //No funcionaif($("div.header")){   //Hago algo}
JQuery siempre retorna un objeto cuando se usa un Selector
Y como ya vimos un objeto evaluado en un if es TRUE.

Debemos usar la propiedad length 
if($("div.header").length){
   //Hago algo}
